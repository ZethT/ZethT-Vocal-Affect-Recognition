Results & Conclusions

Model accuracy comparison, confusion matrix analysis, feature importance, and key takeaways from the classification experiments.

Model Performance

Accuracy Comparison

Each classifier was evaluated on an 80/20 stratified train-test split. Two feature sets were tested: the base four acoustic features (f0, spectral centroid, spectral rolloff, HNR) and the MFCC-augmented set that adds 13 Mel-Frequency Cepstral Coefficients, bringing the total feature count to 17.

Model Base Features (4) + MFCCs (17) Δ Gain
Logistic Regression Deployed 58.6% 91.4% +32.8 pp
Random Forest ~70.0% 90.0% +20.0 pp
XGBoost — 92.8% highest overall
⚡
MFCCs were the decisive feature set. Adding 13 MFCC coefficients raised Logistic Regression accuracy from 58.6% to 91.4% — a 32.8 percentage-point gain. The base acoustic features (f0, spectral centroid, rolloff, HNR) were insufficient on their own; the spectral envelope captured by MFCCs contains the information that separates belt from breathy singing.
Classification Results

Confusion Matrix

The confusion matrix below shows the Logistic Regression classifier's error distribution on the held-out test set. Both classes achieve high recall, confirming that the model does not systematically favor one vocal technique over the other.

Interpretation

The off-diagonal cells represent misclassified samples. Both belt (high energy) and breathy (low energy) classes achieve high precision and recall, indicating the classifier is well-calibrated for both directions of error.

The small number of misclassifications likely corresponds to edge-case recordings — clips where a singer transitions between techniques or where recording conditions produce atypical spectral envelopes.

Because the VocalSet dataset is nearly balanced (~52% belt, ~48% breathy), accuracy is not misleading here — the classifier genuinely discriminates between the two techniques rather than exploiting a class imbalance.

Dimensionality Reduction

PCA — 2D Feature Projection

PCA was applied to the four base acoustic features after z-score normalization to visualize class separability in reduced feature space — and to surface collinearity between features before modeling.

📊

pca_scatter.png

Figure 4. PCA Scatter (PC1 vs. PC2) — base acoustic features, z-score normalized

Interpretation

PC1 captures the shared variance between spectral centroid and spectral rolloff — which correlate at r = 0.89 — effectively collapsing the two redundant brightness features into a single axis. PC2 separates along HNR and f0.

Partial overlap between the belt and breathy clusters in 2D confirms that the four base features alone are insufficient for clean linear separation. This motivates the addition of MFCC features, which encode timbral texture not visible in single spectral statistics.

The collinearity resolved by PCA also explains why spectral centroid and rolloff each contribute only ~4–6% of Random Forest feature importance despite being acoustically meaningful — they carry redundant information.

Discrimination Ability

ROC Curves

ROC curves measure each classifier's trade-off between true positive and false positive rates across all classification thresholds. AUC values close to 1.0 indicate strong discrimination ability regardless of the operating threshold chosen.

Interpretation

Both the Logistic Regression and Random Forest curves hug the upper-left corner, indicating high sensitivity with low false positive rate across thresholds. The AUC for both models exceeds 0.95, confirming that the classifier generalizes well beyond the default 0.5 decision boundary.

The close proximity of the two curves suggests diminishing returns from the more complex Random Forest — the linear decision boundary of Logistic Regression is sufficient given the spectral separability introduced by MFCCs.

The diagonal dashed line represents a random classifier (AUC = 0.5). Both models are far above it, ruling out the possibility that results are driven by class imbalance or overfitting artifacts.

Conclusions

Key Findings

Four central conclusions emerge from the classification experiments.

Finding 01

MFCCs are the decisive feature set

Base acoustic features (f0, centroid, rolloff, HNR) alone were insufficient — Logistic Regression achieved only 58.6% without them, barely above random on a two-class problem. Adding 13 MFCC coefficients raised accuracy to 91.4%, confirming that vocal technique differences are primarily encoded in the spectral envelope shape, not in raw pitch or brightness alone.

Finding 02

A linear model is competitive with ensemble methods

XGBoost achieved the highest accuracy at 92.8%, with Logistic Regression close behind at 91.4% and Random Forest at 90.0%. The 1.4-point gap between the simplest and most complex model is small — once MFCCs are included, the classification boundary is largely linear in the 17-dimensional feature space. Logistic Regression was selected for deployment because it offers nearly the same accuracy with far greater interpretability and a smaller serving footprint.

Finding 03

Spectral centroid and rolloff are redundant

PCA and EDA confirmed a Pearson correlation of r = 0.89 between spectral centroid and spectral rolloff. Both features track the same acoustic property — timbral brightness — and together they contribute less than 10% of Random Forest feature importance. Removing one would have negligible impact on accuracy and would reduce collinearity in the feature matrix.

Finding 04

Vocal technique reliably encodes arousal level

Reaching 92.8% accuracy on VocalSet's belt vs. breathy distinction validates the core thesis: singing technique carries consistent, extractable acoustic markers of vocal arousal. Belt singing (high energy/effort) and breathy singing (low energy/air-dominant) are separable in the MFCC feature space with high confidence, supporting the use of technique-based acoustic features in music recommendation pipelines.
